## Project Features

### User Side
- Register & Login
- View available travel packages
- Book trips with date, people count & flight type
- View booking history
- Cancel pending bookings
- Search travel packages

### Admin Side
- Login as admin
- Manage users (view)
- Add new travel packages
- Edit & delete travel packages
- View all bookings
- Approve or cancel bookings
- Dashboard with statistics
